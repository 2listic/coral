# simple - Vue application with Rete.js

## Project setup

### Build docker image
```
docker build . -t coral-editor
```

### Run image
```
docker run --detach --rm -it -p 8080:8080 coral-editor
```

### Open in browser
```
http://localhost:8080/
```

## To develop just the application
```
npm install && npm run serve
```
